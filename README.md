Age Calculator - CodeAlpha Task 1

This is a web-based **Age Calculator** application developed during my internship at **CodeAlpha**. It allows users to calculate their exact age in years, months, and days by selecting their date of birth.

## ✨ Features
- ✨ **Accurate Calculation:** Computes age based on years, months, and days.
- 📱 **Responsive Design:** Works perfectly on both desktop and mobile screens.
- 🎨 **Clean UI:** Simple and user-friendly interface.
- ✅ **Input Validation:** Handles empty or invalid date selections.

## 🛠️ Tech Stack
- **HTML5:** For the basic structure.
- **CSS3:** For styling and a modern look.
- **JavaScript:** For the calculation logic and DOM manipulation.

## 🚀 How to Run
1. Clone the repository or download the files.
2. Open `index.html` in any modern web browser (Chrome, Edge, Firefox).
3. Select your date of birth and click the **Calculate** button.

## 📂 Project Structure
- `index.html` - The main structure of the web app.
- `style.css` - Styling and layout design.
- `script.js` - Core logic for age calculation.
